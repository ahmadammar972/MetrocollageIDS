Introduction to AI and Data Science

Welcome to the course modules. Please select a lecture below to begin:

  • Lecture 1: Foundations of AI and Data Science

    Topics include the history of Data Science, essential tools and skills, and the AI ecosystem.

  • Lecture 2: AI/Data Science Project Lifecycle

    Topics include the Data Analytics Framework, project lifecycle phases, and Data-Driven Decision Making (DDD).

  • Lecture 3: EDA, Univariate, Bivariate, and Multivariate Analysis

    Topics focus on Exploratory Data Analysis (EDA) and various forms of statistical analysis.

  • Lecture 4: Introduction to Predictive Analytics

    Topics include predictive models, supervised and unsupervised learning, linear regression, and decision trees.

  • Lecture 5: Introduction to Neural Networks and Deep Learning

    Topics include neural network overviews, deep learning concepts, and applications like Face Recognition and NLP.
